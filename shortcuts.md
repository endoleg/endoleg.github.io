# My favorite MS Outlook Shortcuts

Switch to Mail view: Ctrl+1

Switch to Calendar view: Ctrl+2

Go to the Search box: Ctrl+E

Check names: Ctrl+K

Forward: Ctrl+F

Reply to a message: Ctrl+R

Create a new message (when in Mail view): Ctrl+N

Send: Ctrl+Enter
