Mit powercfg.exe /query bekommt man sehr viele Infos!

Oder hier: http://ss64.com/nt/powercfg.html

Set the 'Power Management' to Balanced

powercfg.exe -SETACTIVE SCHEME_BALANCED

Setzt die Einstellung, dass der Monitor im Sperrbildschirm erst nach 18000 Sek / 300 Min / 5 Stunden ausschaltet

powercfg.exe -setacvalueindex SCHEME_BALANCED SUB_VIDEO VIDEOCONLOCK 18000 powercfg.exe -setdcvalueindex SCHEME_BALANCED SUB_VIDEO VIDEOCONLOCK 18000

Set the Lid Close Action power setting to do Nothing

powercfg.exe -setdcvalueindex 381b4222-f694-41f0-9685-ff5bb260df2e 4f971e89-eebd-4455-a8de-9e59040e7347 5ca83367-6e45-459f-a27b-476b1d01c936 000 powercfg.exe -setacvalueindex 381b4222-f694-41f0-9685-ff5bb260df2e 4f971e89-eebd-4455-a8de-9e59040e7347 5ca83367-6e45-459f-a27b-476b1d01c936 000

Apply changes without restart by "re-setting" current power scheme after doing changes ;-) powercfg.exe -SETACTIVE SCHEME_BALANCED

Invoke-Expression ("reg add 'HKCU\Software\IWC' /v SkriptEnergieoptionenNEW /t REG_DWORD /d 0x3 /f")

GUIS:

{381b4222-f694-41f0-9685-ff5bb260df2e} = GUID of "Balanced" power plan

{4f971e89-eebd-4455-a8de-9e59040e7347} = GUID of system power control buttons subgroup

{5ca83367-6e45-459f-a27b-476b1d01c936} = GUID of Lid Close Action power setting

Setting 0=Do Nothing

Setting 1=Sleep

Setting 2=Hibernate

Setting 3=Shutdown
